## üåü Key Features

- **Advanced 3D Graphics:** Built with Three.js featuring procedural neon environments and volumetric lighting.
- **Pro Multiplayer Lobbies:** Create private rooms with 4-letter codes and race up to 4 friends simultaneously.
- **Dynamic Physics:** Custom vehicle physics including drifting, handbraking, and a manual Nitrous (NOS) system.
- **Adaptive AI:** Smart computer-controlled racers that utilize different racing lines and boost strategies based on difficulty.
- **AAA UI Design:** A fully responsive, two-column professional menu system designed for all screen sizes.
- **Real-time Leaderboards:** Accurate position tracking using distance-to-checkpoint tie-breakers.
